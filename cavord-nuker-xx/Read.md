# cavord-nuker-xx/Discord

* This is a mini tool - small in size but fully equipped with all the necessary features to nuke a server.
* A nuke tool built entirely in Python, designed to execute commands quickly.
* The tool only runs on Windows 64-bit.

## 〢 Features

- [x] Auto. ( 2 -> 5 )
- [x] Change name.
- [x] Delete channels.
- [x] Create channels.
- [x] Webhook spam.
- [x] Massban.
- [x] Massrole. (delete -> create)
- [x] Delete emoji & sticker.

## 〢 Tutorial

1. Download the `.exe` file and run`.exe` file 
2. Enter your bot token and the guild ID you want to nuke and use.

> [!NOTE]
> Avoid using the auto and webhook spam feature excessively.
